show(options)

初始化并显示弹框
参数:
名称 类型 描述
options object 初始化参数
Properties
名称 类型 描述
titleContainer jQuery 可选，弹框父层的title
insertContainer jQuery 可选，想要将弹框插入的容器
referenceContainer jQuery 可选，弹框要参考的容器，主要用于获取容器的宽度和位置
addDialogFlagClass string 可选，想要在弹出框中添加的样式类名
toHideContainer jQuery 可选，显示弹框时,要隐藏的容器
title string 可选，弹出框的title,可在content中添加,请参考example
content string 必填，在弹出框中显示的内容
footer string 可选，在弹出框页脚,可在content中添加,请参考example
aside string 可选，在弹出框页脚,可在content中添加,请参考example
close callback 可选，当关闭弹框的回调
closeBefore callback 可选，当关闭弹框前的回调,返回true不关闭弹框,返回false关闭弹框
ready callback 可选，初始化且动画执行完成的回调
render callback 可选，dom节点生成并插入页面时的回调，此时动画未完成
