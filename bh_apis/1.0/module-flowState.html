内置默认值
属性:
名称 类型 描述
defaults object
Properties
名称 类型 描述
width string 流程提示条的宽度
flowStateData object 流程提示的步骤的参数的集合
id string 流程提示的步骤的id号
content string 流程提示的步骤的主要内容
status string 流程提示的步骤的状态，成功：success; 失败：fail; 操作中：operation ; 未开始： not started
statusDescription string 流程提示的步骤的状态的描述，成功，失败，审核中，还没开始（为空）
isShowPop boolean 流程提示的步骤是否展示气泡弹窗
popHtml string 流程提示的步骤展示的气泡弹窗的dom元素,dom元素的样式需要自己写
