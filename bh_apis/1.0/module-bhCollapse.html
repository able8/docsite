配置

(static) $.fn.bhCollapse.defaults

内置默认值
属性:
名称 类型 描述
defaults object
Properties
名称 类型 描述
model string 可选，设置展开面板节点的模式，single/multiple：只展示一个面板节点/可同时展开多个面板节点，默认为'single'
extend boolean 可选，初始展开所有面板节点, 仅在multiple模式下生效, 默认false，
attrs boolean 可选，给面板节点添加自定义属性,并在展开回调(nodeExtend)中返回该数据
data Array 必填，面板节点的配置数组
Properties
名称 类型 描述
title string 必填，面板节点的标题
content Array 必填，面板节点展开项的配置数组
Properties
名称 类型 描述
title string 可选，面板节点展开项的标题
content Array 可选，面板节点展开项的内容文本
extend boolean 可选，设置展开该面板节点，默认false
disabled boolean 可选，设置该面板节点为disabled状态，此时没有Hover效果、无法点击。
nodeExtend function 可选，展开节点的回调
nodeCollapse function 可选，收缩节点的回调
ready function 可选，创建完成的回调
