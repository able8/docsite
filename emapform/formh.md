# emapForm 表格表单

### 初始化参数说明
```js
$('#horizonEditForm').emapForm({
  ...
  model: "t",  //表单布局方式; 只在非只读表单中生效
  cols: 3  //表单布局列数，只在只读表单和表格表单中生效，可选值 1 2 3
  ...
});
```

#### 提示：
model参数含义
* 'h' = horizontal 水平布局
* 'v' = vertical 垂直布局 
* 't' = table 表格布局
