# 只读表单


### 初始化参数说明
在只读表单中，表单布局方式设置无效
```js
$('#horizonEditForm').emapForm({
...
readonly: true,
cols: 3 //表单布局列数，只在只读表单和表格表单中生效，可选值 1 2 3
...
});
```
