方法API

$.emapImport()

属性:
名称 类型 属性 默认值 描述
app String 应用名称
module String 模块名称
page String 页面名称
action String 动作名称
storeId String <optional>
 imexport emap文件类型
preCallback function <optional>
 导入数据弹窗弹出后的回调
importCallback function <optional>
 导入数据成功的回调
closeCallback function <optional>
 导入弹窗关闭的回调
closeConfirmCallback function <optional>
 点击确认关闭按钮的回调
autoClose Boolean <optional>
 false 导入成功后是否自动关闭弹窗
tplUrl String <optional>
 自定义导入模板的下载地址
queryTimes String <optional>
 轮询导入进度间隔
params Object <optional>
 额外参数
Properties
名称 类型 属性 描述
colnames String <optional>
 需要导出的字段的name，多个用逗号分隔，选填
filename String <optional>
 模板文件的名称，不填则默认取模型的名称，选填
readTemplate String <optional>
 下载导入模板请求-自定义的导入读文件服务，实现IImportRead，选填
readImport String <optional>
 获取导入行数和导入请求-自定义的导入读文件服务，实现IImportRead，选填
consts String <optional>
 导入请求-常量字段，多个用逗号分隔，内容为key:value，选填
guids String <optional>
 导入请求-GUID字段，多个用逗号分隔，选填
analyse String <optional>
 导入请求-自定义的导出过程分析服务，实现IImportAnalyse，选填
save String <optional>
 导入请求-自定义的写入保存服务，实现IImportSave，选填
ignoreEmptyRow Boolean <optional>
 是否忽略空行（空格也算空行），true表示读取忽略空行，false表示将空行当成正常记录读取，选填，默认为true
